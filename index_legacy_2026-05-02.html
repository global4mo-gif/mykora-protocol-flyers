06 Usage

Agent-ready prompt

Короткая команда для агента, когда нужно быстро собрать новый интерфейс или визуал в этом стиле.

Prompt

Build this in the MYKORA design system: warm ivory grid or scientific pattern background, deep olive text (#3E3A17), vivid orange as the only strong action accent (#FF7F00), white rounded panels, Unbounded for headings and UI labels, Libertinus Sans for body copy, use logo variants from the MYKORA logo library, use the signature asset for social visuals, keep the tone scientific, warm, practical, and trustworthy.
